HASTE

This was my 2nd project, done as part of the General Assembly coursework. We had just learnt about Node and Express and were told to build an app with them. The framework was quite straightforward. It was like HTML but with a bit of routing and data fetching from external APIs.

Features:

  • Index of orders/deliveries + summary bar
  • Integrated Google Maps
  • Integrated OneMap API for postal-to-coordinates conversion
  • Data stored in MongoDB cloud
  • Staff login using Passportjs

Sorting visualiser

Currently, the sorter runs 4 sorting algorithms: Quick, Merge, Bucket, and Heap. I hope to implement more.

It is much easier, I found, to manipulate the attributes (height, color, etc.) of the nodes rather than the shift them around. They get 'lost' sometimes when you have clone them. Nevertheless, the process of studying the algorithms and visualising them was extremely enjoyable and satisfying.
